> This module contains the commons code from hapi projetcs

## Installation

```sh
$ npm install --save don-nodejs-commons-hapi
```

## Usage

```js
var donNodejsCommonsHapi = require('don-nodejs-commons-hapi');

donNodejsCommonsHapi('Rainbow');
```
